## Preparation

你需要安装[node](https://nodejs.org/) and  [git](https://git-scm.com/) . 
项目基于 [ES2015+](https://es6.ruanyifeng.com/), [vue](https://cn.vuejs.org/index.html), [vuex](https://vuex.vuejs.org/zh-cn/), [vue-router](https://router.vuejs.org/zh-cn/), [vue-cli](https://github.com/vuejs/vue-cli) , [axios](https://github.com/axios/axios) and [element-ui](https://github.com/ElemeFE/element), 所有请求数据使用模拟 [Mock.js](https://github.com/nuysoft/Mock).


## 准备运行

```bash

# install dependency
npm install

# develop
npm run dev
```



## 构建

```bash
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
```

## 其他命令

```bash
# 预览发布环境效果
npm run preview

# 预览发布环境效果+静态资源分析
npm run preview -- --report

# 代码格式化eslint检测
npm run lint

# 代码自动格式化eslint
npm run lint -- --fix
```
